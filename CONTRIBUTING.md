# Contributing

Thanks for interest in **Adrift**.

## Setup

See the [README](README.md). Static scene + optional `npm run build:js` for viewer edits.

## Conventions

- Edit readable source in `js/`; regenerate `jsm/` with `npm run build:js`.
- Prefer verifying against art `/adrift/experience/` after assemble.
- Never commit secrets.

## Pull requests

Focused diffs; describe UI or weather-HUD impact. Contact via GitHub issues.
